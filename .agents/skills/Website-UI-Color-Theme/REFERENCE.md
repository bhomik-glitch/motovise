# Supplemental Color Reference Guide

*   **Real-World Example:** Lemon Squeezy applies the 60-30-10 rule by using purple for 60% of the UI, yellow for 30%, and white as the 10% accent color [3].
*   **Brand Color Adaptation:** Don't be afraid to adjust provided brand colors for accessibility [6]. You can rotate colors slightly on the color wheel to find analogous combinations, or pick complementary colors from across the wheel, such as pairing bright yellow with a complementary turquoise [11].
*   **Dark Mode Considerations:** Do not just invert your light mode colors [7, 9]. Dark mode backgrounds need dark colors, while foregrounds need slightly lighter elements to create depth since shadows aren't visible [12]. Dark colors need more contrast between them to be visibly different [7]. Use light grays instead of pure white for regular text to reduce eye strain, reserving pure white for only the most important actions or logos [7, 9].
