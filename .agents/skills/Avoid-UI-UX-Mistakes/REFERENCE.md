# Supplemental Mistakes Reference Guide

*   **Poor Icon Implementation:** Avoid mismatched icons by sticking to a single library (like Feather icons or Phosphor icons) so line width, fill, and style remain consistent [19]. Ensure SVG formats are used for high quality [19]. Use tooltips for ambiguous icons to improve user onboarding [19]. It is okay to use different icon styles only if they are placed in visually separate areas of the design [19].
*   **Overdesigned Charts:** Prioritize legibility over aesthetics [20]. Always include vertical axes and ensure data points are easy to read instead of prioritizing visually stylized but unreadable graphs [18, 20].
